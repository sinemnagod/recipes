# 🍕 Recipes Website

A simple recipe website built with pure HTML and CSS — my very first frontend project!

🔗 **Live Site:** [sinemnagod.github.io/recipes](https://sinemnagod.github.io/recipes)

---

## About

This was my first-ever web development project, built to learn the fundamentals of HTML and CSS. No frameworks, no JavaScript, just the basics.

The site features a home page with a hero section, recipe cards, a quote, and a newsletter signup footer. Each recipe has its own dedicated page.

## Recipes Included

- 🥐 Cinnamon Roll
- 🍕 Pizza
- 🥗 Pasta Salad

## What I Learned

- HTML document structure and semantic elements
- CSS styling, layout, and typography
- Linking pages and assets across folders
- Deploying a static site with GitHub Pages
